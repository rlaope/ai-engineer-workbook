## 1. 매니폴드 가정

### 1.1 정의

*매니폴드 (Manifold)* 는 *지역적으로 유클리드 공간처럼 보이는* 공간입니다. 지구 표면이 가까이서 보면 평면처럼 보이지만 전체적으로는 곡면인 것과 같습니다.

ML 의 *매니폴드 가정*: 고차원 데이터는 *낮은 차원의 매니폴드* 위에 놓여 있다.

### 1.2 함의

이 가정이 사실이라면:

- *차원 축소* (PCA, t-SNE, UMAP) 가 의미 있는 표현을 추출
- *생성 모델* (VAE, GAN, Diffusion) 이 매니폴드를 학습 가능
- *데이터 증강* 도 매니폴드 위에서만 의미 있음

### 1.3 챕터 정리

매니폴드 가정은 ML 의 거의 모든 표현 학습·생성 모델의 수학적 기반입니다.

## 2. 정보 기하

### 2.1 Fisher 정보 행렬

확률 분포 $p(x; \theta)$ 의 *Fisher 정보 행렬*:

$$F_{ij}(\theta) = E\left[\frac{\partial \log p}{\partial \theta_i} \frac{\partial \log p}{\partial \theta_j}\right]$$

직관: *파라미터 변화에 대한 분포의 민감도* 를 측정.

### 2.2 자연 그래디언트

표준 그래디언트 디센트는 *유클리드 거리* 로 파라미터 공간을 측정합니다. 그러나 *분포 공간의 자연스러운 거리* 는 KL 발산이며, Fisher 정보 행렬이 그 *지역 메트릭* 입니다.

자연 그래디언트:

$$W_{t+1} = W_t - \eta F^{-1} \nabla L$$

이론적으로 *훨씬 빠른 수렴* 을 보장하지만, $F^{-1}$ 계산이 *대규모 신경망에서 비현실적* 입니다. K-FAC, Shampoo 같은 근사가 사용됩니다.

### 2.3 챕터 정리

정보 기하는 *분포 공간의 자연스러운 메트릭* 을 정의하며, 자연 그래디언트는 그 응용입니다.

## 3. 매니폴드 학습 알고리즘

### 3.1 PCA — 선형 매니폴드

PCA 는 *데이터의 분산이 큰 선형 부분공간* 을 매니폴드로 가정합니다. 매니폴드가 *비선형* 이면 PCA 는 부적합.

### 3.2 t-SNE, UMAP — 비선형 매니폴드

*비선형 매니폴드* 위의 데이터를 *2D-3D 시각화* 에 적합한 형태로 투영합니다.

```python
from sklearn.manifold import TSNE
embedding = TSNE(n_components=2).fit_transform(X)
```

t-SNE 는 *지역 구조* 를 잘 보존하지만 *전역 구조* 는 왜곡 가능. UMAP 은 *전역 구조* 도 비교적 보존.

### 3.3 챕터 정리

매니폴드 학습 알고리즘은 *PCA (선형) → t-SNE/UMAP (비선형)* 으로 진화했습니다.
