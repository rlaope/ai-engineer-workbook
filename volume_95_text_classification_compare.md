# Volume 95 — 텍스트 분류 4 접근 비교

> 이 권이 끝나면 새 분류 작업을 받았을 때 *4 가지 접근 중 어떤 것을 시도할지* 를 30 초 안에 결정할 수 있게 됩니다.

## 목적

같은 텍스트 분류 작업을 *표현 모델 직접 분류*·*작업 특화 fine-tuned 모델*·*임베딩 + 단순 분류기*·*생성 모델 zero/few-shot* 의 4 가지 방법으로 풀 수 있습니다. 각 방법은 데이터 양·정확도·지연·비용에서 다른 트레이드오프를 가집니다. 이 권은 4 접근의 강약을 같은 데이터셋(영화 리뷰 등) 으로 비교 실험합니다.

## 선수 지식

- Volume 12, 33, 34, 39 완료
- 외부 지식: 모델 비교 실험의 일반 절차

## 학습 결과

이 권을 마치면 다음을 할 수 있게 됩니다.

1. 4 가지 접근의 *입력·출력·학습 비용·추론 비용* 을 표로 비교할 수 있습니다.
2. 데이터 양에 따른 권장 접근을 제시할 수 있습니다.
3. 같은 데이터셋으로 4 접근을 모두 구현·비교할 수 있습니다.
4. *작은 모델로 충분한가 vs LLM 이 필요한가* 를 데이터로 판단할 수 있습니다.
5. 산업 사용 사례별 표준 선택 매트릭스를 갖습니다.

## 챕터 목차

1. **텍스트 분류 작업의 정의와 평가**
2. **접근 1 — 표현 모델(BERT) 직접 분류**
3. **접근 2 — 작업 특화 사전학습 모델** (예: DistilBERT-SST2)
4. **접근 3 — 임베딩 + Logistic Regression / k-NN**
5. **접근 4 — 생성 모델 Zero/Few-shot**
6. **데이터 양별 권장 접근** — 0/100/1K/10K/100K
7. **비용·지연·정확도 비교 실험**
8. **선택 의사결정 매트릭스**

## 자가점검 키워드

`표현 모델`, `작업 특화 모델`, `임베딩 분류`, `Zero/Few-shot`, `데이터 효율성`, `비용 비교`, `의사결정 매트릭스`, `벤치마크`
