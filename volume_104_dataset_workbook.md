# Volume 19 — 데이터셋 종합 실습 워크북

> 이 권이 끝나면 *MNIST·CelebA·Tiny ImageNet·SST-2·SQuAD* 같은 표준 데이터셋을 *처음부터 끝까지* 다뤄 본 경험을 갖게 됩니다.

## 목적

워크북의 다른 권들은 *개념·도구·기법*을 다룹니다. 이 권은 *그 모두를 한 데이터셋 위에서 통합 실습*하는 워크북입니다. 분류·생성·시퀀스·검색 각 분야에서 가장 표준적인 데이터셋을 골라, *데이터 로딩 → 전처리 → 모델 → 학습 → 평가 → 배포 미니 단계까지* 를 한 권 안에 넣습니다. 이 권은 워크북을 끝까지 본 사람이 *처음으로 자기 손으로 한 학습 사이클을 통째로 돌리는* 무대입니다.

## 선수 지식

- 모든 핵심 트랙(D, E, F, H) 의 기본 권 완료
- 외부 지식: GPU 사용 가능 환경

## 학습 결과

이 권을 마치면 다음을 할 수 있게 됩니다.

1. PyTorch `DataLoader` + 표준 데이터셋 로딩 코드를 외울 수 있습니다.
2. 한 데이터셋에 대해 학습 → 평가 → 시각화의 전체 사이클을 30 분 안에 다시 돌릴 수 있습니다.
3. 분류·생성·시퀀스·검색 각각의 표준 워크플로를 비교할 수 있습니다.
4. 자신의 노트북/실험 양식을 데이터셋과 함께 묶어 둘 수 있습니다.
5. 다른 권에서 배운 개념의 *효과를 데이터로 확인* 할 수 있습니다.

## 챕터 목차

1. **MNIST** — 분류의 첫 사이클 (MLP → CNN → 비교)
2. **Fashion-MNIST·CIFAR-10** — 비전 분류의 단계
3. **Tiny ImageNet** — ResNet 미세조정
4. **CelebA** — 얼굴 데이터·DCGAN 실습
5. **SST-2** — 텍스트 분류 4 접근 비교 (Vol 102 와 결합)
6. **SQuAD** — 질의응답 + 추출
7. **MS MARCO** — 임베딩 검색 평가
8. **Stable Diffusion 미세조정 — Textual Inversion·LoRA**
9. **자기 데이터셋 추가하기 — 가이드라인**
