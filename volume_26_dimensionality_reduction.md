## 1. PCA — 선형 축소

이미 Vol 9 (선형대수 2) 에서 다뤘으므로 여기서는 *차원 축소 도구* 로서의 PCA 만 정리.

```python
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt

pca = PCA(n_components=2)
X_2d = pca.fit_transform(X_high_dim)
plt.scatter(X_2d[:, 0], X_2d[:, 1])
print(pca.explained_variance_ratio_.sum())   # 설명력
```

장점: *빠르고 결정론적*, *전역 구조 보존*.
단점: *비선형 매니폴드* 에 약함.

---

## 2. t-SNE

### 2.1 발상

*고차원의 거리 분포* 와 *2D 의 거리 분포* 가 *가능한 한 비슷* 해지도록 2D 좌표를 학습.

```python
from sklearn.manifold import TSNE
X_2d = TSNE(n_components=2, perplexity=30).fit_transform(X)
```

### 2.2 강약

- 강점: *지역 구조 보존* 우수, 시각화에 가장 자주 사용
- 약점: *전역 구조 왜곡*, *느림 (O(n²))*, *비결정론적*, *perplexity 민감*

### 2.3 함정

- t-SNE 그림에서 *군집 사이 거리* 는 *의미가 없음*
- 같은 데이터를 두 번 돌리면 다른 그림
- *perplexity* (보통 5-50) 선택이 중요

---

## 3. UMAP

t-SNE 의 후계자. *지역 + 전역 구조 모두 비교적 보존*, *더 빠름*, *수학적 기반 (위상수학) 명확*.

```python
import umap
reducer = umap.UMAP(n_neighbors=15, min_dist=0.1)
X_2d = reducer.fit_transform(X)
```

장점: t-SNE 보다 *빠르고 안정적*. 큰 데이터에 적용 가능.

현재 *임베딩 시각화의 새 표준* 으로 자리잡고 있습니다.

---

## 4. 비교

```
+----------+-----------+-----------+-----------+
|          | PCA       | t-SNE     | UMAP      |
+----------+-----------+-----------+-----------+
| 종류     | 선형      | 비선형    | 비선형    |
| 속도     | 매우 빠름 | 느림      | 빠름      |
| 결정론   | Yes       | No        | No        |
| 전역구조 | 우수      | 약함      | 보통      |
| 지역구조 | 약함      | 우수      | 우수      |
| 큰 데이터| 가능      | 어려움    | 가능      |
+----------+-----------+-----------+-----------+
```

선택 가이드:

- *빠른 첫 시각화* → PCA
- *지역 구조 강조* → t-SNE
- *대규모 + 전역+지역 균형* → UMAP

---

## 권 정리

- PCA = 선형·빠름·전역 보존
- t-SNE = 비선형·지역 보존·시각화 표준
- UMAP = t-SNE 후계·빠름·균형
- 차원 축소 그림 해석 시 *알고리즘별 함정* 을 알아야 정확히 읽음

가장 기억할 한 줄: **"임베딩 시각화는 UMAP 부터 시도하고, 빠른 확인은 PCA, 지역 구조 깊이 보려면 t-SNE."**
