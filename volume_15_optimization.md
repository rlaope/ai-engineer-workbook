# Volume 15 — 최적화 이론

> 이 권이 끝나면 *학습은 비볼록 최적화* 라는 한 문장이 던지는 모든 함의를 이해하게 됩니다.

## 목적

신경망 학습은 본질적으로 *손실 함수의 최솟값을 찾는 최적화 문제* 입니다. 그러나 손실 함수는 *비볼록* 이라 *전역 최솟값을 보장하지 않으며*, 그럼에도 SGD 같은 단순한 알고리즘이 *충분히 좋은 해* 를 찾는다는 것이 딥러닝의 미스터리이자 핵심입니다. 이 권은 볼록·비볼록 최적화의 큰 그림과 KKT·라그랑지 같은 도구를 다룹니다.

## 1. 볼록 vs 비볼록

### 1.1 볼록 함수의 정의

함수 f 가 *볼록 (convex)* 이라는 의미는, 임의의 두 점 x, y 와 0 ≤ λ ≤ 1 에 대해:

$$f(\lambda x + (1-\lambda) y) \leq \lambda f(x) + (1-\lambda) f(y)$$

직관: *함수 그래프 위의 두 점을 잇는 선분* 이 *함수 그래프 위에 위치*.

볼록 함수의 핵심 성질:

- *지역 최솟값 = 전역 최솟값*
- *경사하강법이 항상 전역 최솟값에 수렴* (적절한 학습률 하에)

### 1.2 비볼록 함수

신경망의 손실 함수는 거의 항상 *비볼록* 입니다.

- 여러 *지역 최솟값* 존재
- *안장점 (saddle point)* — 어떤 방향으로는 최솟값, 다른 방향으로는 최댓값
- *고원 (plateau)* — 그래디언트가 거의 0 인 평평한 영역

이 모든 함정에도 불구하고 SGD 가 *충분히 좋은 해* 를 찾는 것이 딥러닝의 *경험적 발견* 입니다.

### 1.3 챕터 정리

볼록 함수는 *지역 = 전역 최솟값* 의 보장을 가지지만, 신경망 손실은 비볼록입니다. 비볼록에서 SGD 가 작동하는 이유는 *완전히 이론적으로 설명되지 않은* 영역입니다.

## 2. 라그랑지 승수와 KKT

### 2.1 제약 최적화

다음 형태의 문제:

$$\min f(x) \quad \text{subject to} \quad g_i(x) \leq 0, \quad h_j(x) = 0$$

라그랑지 함수:

$$L(x, \lambda, \mu) = f(x) + \sum_i \lambda_i g_i(x) + \sum_j \mu_j h_j(x)$$

### 2.2 KKT 조건

최적해에서 다음 조건이 만족됩니다 (정칙성 조건 하에).

1. *정상 조건* — $\nabla_x L = 0$
2. *원시 가능성* — $g_i(x) \leq 0$, $h_j(x) = 0$
3. *쌍대 가능성* — $\lambda_i \geq 0$
4. *상보 슬랙* — $\lambda_i g_i(x) = 0$

이 조건들이 *제약 최적화 문제의 표준 도구* 입니다.

### 2.3 ML 에서의 등장

- **SVM** — 최대 마진 분류기 도출에 KKT 사용
- **L1 정칙화 (Lasso)** — 라그랑지 형태로 유도
- **GAN 의 minimax** — 두 플레이어 게임의 균형점

### 2.4 챕터 정리

라그랑지·KKT 는 제약 최적화의 표준 도구이며, SVM·Lasso·GAN 같은 ML 모델의 수학적 기반입니다.

## 3. 1차 vs 2차 최적화

### 3.1 1차 방법

*그래디언트만* 사용:

- SGD, 모멘텀, Adam, AdamW

장점: *계산 비용 낮음*, *대규모 모델에 적용 가능*
단점: *수렴 속도가 느림*, *학습률 튜닝 필요*

### 3.2 2차 방법

*헤시안* (2차 미분) 도 사용:

- Newton 법, Quasi-Newton (BFGS, L-BFGS)

장점: *빠른 수렴*, *학습률 튜닝 부담 적음*
단점: *헤시안 계산·저장 비용 막대* (n × n 행렬)

신경망에서는 헤시안이 *수십억 × 수십억* 크기라 *직접 계산 불가능*. 따라서 1차 방법이 표준.

### 3.3 자연 그래디언트

*Fisher 정보 행렬* 을 헤시안의 근사로 사용. K-FAC, Shampoo 같은 알고리즘이 이 발상의 응용. 산업에서는 *비용 대비 효과* 가 모호해 잘 사용되지 않지만, 일부 연구에서는 *훨씬 빠른 수렴* 을 보고합니다.

### 3.4 챕터 정리

1차 방법이 *대규모 신경망의 표준*, 2차 방법은 *계산 비용 때문에 거의 사용되지 않음*. 자연 그래디언트는 중간 후보.

## 4. 비볼록에서 SGD 가 작동하는 이유

### 4.1 경험적 관찰

신경망 손실 풍경에는:

- *지역 최솟값들이 대부분 비슷한 손실 값* 을 가짐
- *나쁜 지역 최솟값* 이 드물어 보임
- 안장점이 *그래디언트의 추계적 노이즈* 로 빠져나갈 수 있음

이는 *고차원 최적화의 특수 현상* 으로 추정됩니다.

### 4.2 과매개화의 효과

신경망 파라미터 수가 *데이터 샘플 수보다 큰 경우* (overparametrized), 손실 풍경이 *접속된 손실 평지* 로 단순화된다는 가설이 있습니다 (lottery ticket·neural tangent kernel 등 분야).

### 4.3 챕터 정리

비볼록에서 SGD 의 성공은 *고차원의 특수성·과매개화* 로 부분적으로 설명되지만, *완전한 이론* 은 아직 미해결 문제입니다.

- 볼록 vs 비볼록 — 신경망 손실은 비볼록

- 1차 vs 2차 — 신경망은 1차가 표준
