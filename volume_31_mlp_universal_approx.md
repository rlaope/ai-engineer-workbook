# Volume 31 — MLP와 유니버설 근사

> 이 권이 끝나면 신경망이 *어떤 연속 함수도 임의의 정확도로 근사할 수 있다* 는 정리의 의미와 한계를 모두 설명할 수 있게 됩니다.

## 목적

*유니버설 근사 정리 (Universal Approximation Theorem)* 는 *충분히 큰 신경망이 어떤 연속 함수도 임의 정확도로 근사 가능* 하다고 말합니다. 이 정리는 신경망의 표현력에 대한 강력한 보장이지만, *실용성에 대한 보장은 아닙니다*. 이 권은 그 차이를 다집니다.

## 1. 유니버설 근사 정리

### 1.1 진술

*숨김 층이 하나라도, 충분한 뉴런이 있다면, 어떤 연속 함수도 임의 정확도로 근사 가능*.

수학적으로 (Cybenko 1989, Hornik 1991):

$$\forall f \in C[0,1]^n, \forall \epsilon > 0, \exists \hat{f} = \sum w_i \sigma(a_i^T x + b_i) : \|f - \hat{f}\| < \epsilon$$

### 1.2 의미와 한계

**의미**: 신경망의 *표현 능력은 본질적으로 무제한*.

**한계**:
- *얼마나 큰 네트워크가 필요한지* 보장 없음 (지수적일 수 있음)
- *학습 가능 여부* 보장 없음 — 이론은 존재하지만 SGD 가 찾을 수 있다는 보장 없음
- *일반화 보장 없음* — 학습 데이터에 과적합할 수 있음

### 1.3 챕터 정리

유니버설 근사 정리는 *표현력의 보장* 이지 *실용성의 보장* 이 아닙니다.

## 2. 깊이 vs 너비

### 2.1 단층 vs 다층

같은 함수를 근사할 때:

- *단층 (1 hidden layer)* — 이론적으로 가능, 실제로는 *매우 많은 뉴런 필요* (지수적)
- *다층 (deep)* — *훨씬 적은 뉴런으로 같은 함수 표현*

이를 *깊이의 효율성 (Depth Efficiency)* 이라 합니다 (Telgarsky 2016).

### 2.2 깊이의 다른 장점

- *계층적 표현* — 저수준 → 고수준 추상화
- *재사용 가능한 부품* — 중간 표현이 여러 곳에 활용
- *적은 파라미터*

### 2.3 챕터 정리

이론적으로 단층도 충분하지만, *실용적으로 깊이가 결정적* 입니다.

---

## 3. 깊은 네트워크의 어려움

### 3.1 그래디언트 소실/폭발

깊은 네트워크에서 *역전파 시 그래디언트가 곱해지면서* 0 또는 무한대로 발산합니다.

방어책:
- *적절한 초기화* (He, Xavier)
- *배치 정규화*
- *잔차 연결 (Residual Connection)*
- *그래디언트 클리핑*

### 3.2 학습 어려움

깊은 네트워크는 *손실 풍경이 복잡* 해 학습이 불안정. 2010 년대 이전에는 *3-4 층 이상의 신경망 학습이 매우 어려웠음*.

해결: ReLU·BatchNorm·ResNet 같은 기술적 혁신이 *수십 층 학습* 을 가능하게 만듦.

### 3.3 챕터 정리

깊이는 *표현력에 도움* 이지만 *학습이 어려워짐*. 2010 년대의 기술적 혁신이 깊은 네트워크 학습을 실용화했습니다.

## 권 정리

- 유니버설 근사 = 표현력 보장 (실용성 X)
- 깊이 효율성 = 같은 함수를 적은 뉴런으로
- 깊이의 학습 어려움 = ReLU·BN·ResNet 으로 해결

가장 기억할 한 줄: **"신경망이 어떤 함수도 표현할 수 있다는 것과 학습할 수 있다는 것은 다르다."**
