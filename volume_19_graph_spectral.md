## 1. 그래프의 행렬 표현

### 1.1 인접 행렬

$n$ 개 노드 그래프의 *인접 행렬* $A$:

$$A_{ij} = \begin{cases} 1 & \text{if } i \sim j \\ 0 & \text{otherwise} \end{cases}$$

```python
import numpy as np

# 4 노드 그래프: 0-1, 1-2, 2-3
A = np.array([
    [0, 1, 0, 0],
    [1, 0, 1, 0],
    [0, 1, 0, 1],
    [0, 0, 1, 0],
])
```

### 1.2 차수 행렬

각 노드의 차수 (연결된 간선 수) 를 대각선에 가지는 대각 행렬:

```python
D = np.diag(A.sum(axis=1))
```

### 1.3 라플라시안

*그래프 라플라시안*:

$$L = D - A$$

라플라시안의 성질:
- *대칭, 양의 준정부호 (positive semi-definite)*
- 가장 작은 고유값은 0 (대응 고유벡터는 모두 1)
- *고유값의 분포가 그래프의 구조* 를 반영

### 1.4 챕터 정리

그래프는 *인접 행렬·차수 행렬·라플라시안* 의 세 행렬로 표현됩니다.

## 2. 스펙트럴 분석

### 2.1 라플라시안의 고유 분해

$$L = U \Lambda U^T$$

- $\Lambda$: 고유값 (오름차순)
- $U$: 고유벡터 (행렬)

이 분해가 *그래프의 푸리에 변환* 역할을 합니다. 신호 처리에서 *시간 도메인 → 주파수 도메인* 변환이 푸리에 변환이라면, 그래프에서는 *노드 도메인 → 스펙트럴 도메인* 변환이 라플라시안 고유 분해입니다.

### 2.2 의미

- 작은 고유값 → *부드러운 신호* (이웃 노드 값이 비슷)
- 큰 고유값 → *거친 신호* (이웃 노드 값이 다름)

### 2.3 챕터 정리

라플라시안의 고유 분해는 *그래프의 푸리에 변환* 이며, GNN 의 수학적 기반입니다.

## 3. Graph Convolutional Network (GCN)

### 3.1 정의

GCN 의 한 층:

$$H^{(l+1)} = \sigma(\hat{A} H^{(l)} W^{(l)})$$

- $\hat{A}$: 정규화된 인접 행렬 ($\hat{A} = D^{-1/2}(A+I)D^{-1/2}$)
- $H^{(l)}$: $l$ 층의 노드 특성
- $W^{(l)}$: 학습 가능 가중치
- $\sigma$: 활성화 함수

직관: *각 노드가 이웃의 특성을 가중평균* 해 자기 표현을 갱신.

### 3.2 NumPy 미니 GCN

```python
import numpy as np

def gcn_layer(A_hat, H, W):
    return np.maximum(0, A_hat @ H @ W)   # ReLU

# A: 인접 행렬, H0: 초기 특성
A = np.array([[0,1,0],[1,0,1],[0,1,0]], dtype=float)
A_hat = A + np.eye(3)
D = np.diag(A_hat.sum(axis=1))
D_inv_sqrt = np.diag(1/np.sqrt(np.diag(D)))
A_hat_norm = D_inv_sqrt @ A_hat @ D_inv_sqrt

H0 = np.random.randn(3, 5)   # 3 노드, 5 차원 특성
W = np.random.randn(5, 4)
H1 = gcn_layer(A_hat_norm, H0, W)
print(H1.shape)   # (3, 4)
```

### 3.3 챕터 정리

GCN 은 *이웃 정보의 가중평균 + 학습 가능 변환* 을 반복하는 구조입니다.

## 4. 메시지 패싱 GNN

### 4.1 일반 프레임

GraphSAGE·GAT 같은 *메시지 패싱 (Message Passing)* GNN 은 다음 흐름:

1. *메시지 생성* — 각 노드가 이웃에게 메시지 전송
2. *집계* — 받은 메시지를 노드별로 합침 (sum, mean, max)
3. *갱신* — 자기 표현을 메시지로 갱신

### 4.2 GAT — 어텐션 GNN

GraphAttention Network 는 *어텐션 메커니즘* 으로 *어떤 이웃이 더 중요한가* 를 학습합니다. Transformer 의 어텐션과 같은 사고.

### 4.3 챕터 정리

메시지 패싱 GNN 은 *유연한 일반 프레임* 이며, 어텐션 (GAT) 등 다양한 변형이 있습니다.
