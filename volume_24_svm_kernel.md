## 1. 최대 마진 분류기

### 1.1 직관

두 클래스를 분리하는 *직선 (또는 초평면)* 은 무수히 많습니다. SVM 은 *두 클래스 사이의 빈 공간 (마진) 이 가장 큰* 분리선을 선택합니다.

```
    + +
   + + +     ← 양성 클래스
  + + + +
─────────── ← 결정 경계
   |||||      ← 마진 (지지 벡터로 결정)
─────────── ← 결정 경계
  - - - -
   - - -      ← 음성 클래스
    - -
```

마진이 클수록 *일반화 성능이 좋다* 는 것이 직관적·수학적 결과입니다.

### 1.2 수학

$$\min \frac{1}{2} \|w\|^2 \quad \text{subject to} \quad y_i (w^T x_i + b) \geq 1$$

라그랑지·KKT 로 풀면 *지지 벡터 (Support Vector)* 만 결정 경계에 영향을 미친다는 결과가 나옵니다 (대부분의 점은 무관).

### 1.3 챕터 정리

SVM 의 핵심 직관은 *마진 최대화* 이며, *지지 벡터만 결정 경계를 정의* 한다는 우아한 결과를 가집니다.

---

## 2. 소프트 마진

실제 데이터는 선형 분리가 안 될 때가 많습니다. *슬랙 변수* 를 도입해 *일부 오분류* 를 허용:

$$\min \frac{1}{2} \|w\|^2 + C \sum_i \xi_i$$

`C` 가 클수록 *오분류에 강한 페널티* (마진 작아도 정확도 우선), 작을수록 *마진 크게* (오분류 허용).

---

## 3. 커널 트릭

### 3.1 발상

데이터가 *비선형 분리* 가 필요할 때, *고차원 공간으로 변환 후 선형 분리* 가 가능합니다. 그러나 *명시적 변환은 비용이 큼*.

커널 트릭: *변환된 공간의 내적만 직접 계산* — 변환 자체는 하지 않음.

$$K(x, y) = \phi(x)^T \phi(y)$$

### 3.2 표준 커널

- **Linear** — $K(x, y) = x^T y$
- **Polynomial** — $K(x, y) = (x^T y + c)^d$
- **RBF (Gaussian)** — $K(x, y) = \exp(-\gamma \|x - y\|^2)$
- **Sigmoid** — $K(x, y) = \tanh(\alpha x^T y + c)$

RBF 가 가장 일반적이며, 무한 차원 변환에 해당.

### 3.3 사용

```python
from sklearn.svm import SVC

clf = SVC(kernel='rbf', C=1.0, gamma='scale').fit(X, y)
```

### 3.4 챕터 정리

커널 트릭은 *명시적 고차원 변환 없이 내적만 계산* 하는 우아한 발상이며, 비선형 SVM 을 가능하게 합니다.

---

## 4. SVM 이 적합한 시점

- 작은 데이터 (수천-수만 건)
- 고차원 입력 (텍스트 분류 등)
- 명확한 마진이 있는 문제
- 해석보다 정확도 우선

신경망이 약한 영역과 *상보적* 입니다.

## 권 정리

- 최대 마진 = SVM 의 핵심
- 지지 벡터만 결정 경계 정의
- 소프트 마진 (C 파라미터) = 실제 데이터 처리
- 커널 트릭 = 비선형 분리의 우아한 해법

가장 기억할 한 줄: **"커널 트릭은 변환을 하지 않고 변환 결과의 내적을 계산하는 천재적 발상이다."**
