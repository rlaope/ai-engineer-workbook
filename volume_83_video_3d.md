# Volume 83 — 비디오·3D 생성

> 이 권이 끝나면 *왜 비디오 생성이 이미지 생성보다 훨씬 어려운가* 를 시간 일관성 관점에서 설명할 수 있게 됩니다.

## 목적

비디오·3D 생성은 *이미지의 자연스러운 확장* 처럼 보이지만, *시간 일관성* 과 *3D 구조* 라는 추가 도전을 동반합니다. Sora·Veo·Runway·NeRF·Gaussian Splatting 같은 모델의 본질을 다집니다.

## 선수 지식

- Volume 80 완료

## 학습 결과

1. 비디오 생성의 *시간 일관성* 도전을 이해합니다.
2. Sora·Veo·Runway 같은 모델의 발상을 알 수 있습니다.
3. NeRF 의 *3D 장면 표현* 발상을 안다.
4. Gaussian Splatting 의 *NeRF 가속* 메커니즘을 설명할 수 있습니다.

---

## 1. 비디오 생성의 도전

### 1.1 시간 일관성

연속된 프레임에서 *같은 캐릭터·물체가 일관되게* 유지되어야 함. 이미지 생성은 *프레임 사이의 일관성을 신경쓰지 않음* — 그래서 단순히 프레임을 따로 생성하면 *깜빡거리는 영상*.

### 1.2 모션의 자연스러움

물리 법칙·자연스러운 움직임. 단순한 보간으로는 *기괴한 결과*.

### 1.3 데이터·계산 비용

10 초 비디오 = 240 프레임. 각 프레임이 이미지. *이미지 생성의 240 배 비용* + 시간 일관성 비용.

---

## 2. 비디오 디퓨전 모델

### 2.1 Sora (OpenAI, 2024)

*Spacetime Patch* — 비디오를 *3D 패치 (시간 + 공간)* 로 자르고 디퓨전.

특징:
- 1 분 길이 비디오
- 1080p 해상도
- 강한 물리 시뮬레이션

폐쇄형. API 접근만.

### 2.2 Veo (Google, 2024)

Sora 와 비슷한 능력. Vertex AI 통합.

### 2.3 Runway Gen-3

상용 비디오 생성의 선두. 영화·광고 산업 채택.

### 2.4 오픈 모델

- **Stable Video Diffusion** — 짧은 클립 (4 초 정도)
- **CogVideo** — 중국 발표
- **HunyuanVideo** — 중국, 오픈

---

## 3. NeRF — 3D 장면 표현

### 3.1 발상

여러 각도의 *2D 사진들* 로부터 *3D 장면을 신경망에 인코딩*. 신경망은 *(x, y, z, 시점) → (RGB, 밀도)* 매핑.

### 3.2 결과

새로운 시점에서 *사실적 렌더링*. *3D 메쉬 없이* 가능.

단점: *학습이 매우 느림* (씬마다 수 시간), *렌더링도 느림*.

---

## 4. Gaussian Splatting (2023)

### 4.1 발상

NeRF 의 *신경망 표현 대신 수십만 개의 3D Gaussian* 으로 장면 표현.

### 4.2 효과

- 학습 속도 *10-100 배 빠름*
- 렌더링 *실시간 가능* (60 FPS+)
- 같은 시각 품질

3D 생성·재구성의 *새 표준* 으로 빠르게 자리잡음.

---

## 5. 3D 생성 모델

- **TripoSR** (2024) — 한 장 사진에서 3D 메쉬
- **Trellis** (Microsoft, 2024) — 텍스트→3D
- **Meshy·Genie** — 상용 3D 생성

품질이 *게임·VR·E-Commerce 에서 사용 가능* 수준에 도달 중.

---

## 6. 비디오·3D 의 산업 응용

- **영화·광고** — 사전 시각화·VFX
- **게임** — 자산 생성
- **E-Commerce** — 360° 상품
- **부동산** — 가상 투어
- **교육** — 시각화 자료
- **로보틱스** — 시뮬레이션 데이터

## 권 정리

- 비디오 생성 = 이미지 + 시간 일관성 + 모션
- Sora·Veo·Runway = 폐쇄형 SOTA
- Stable Video Diffusion·HunyuanVideo = 오픈 옵션
- NeRF = 신경망 3D 표현, 느림
- Gaussian Splatting = 10-100 배 빠른 NeRF 대안
- TripoSR·Trellis = 텍스트/이미지→3D

가장 기억할 한 줄: **"비디오 생성의 본질은 이미지 + 시간 일관성이며, 3D 생성은 NeRF 에서 Gaussian Splatting 으로 패러다임이 옮겨가고 있다."**
