## 1. PAC 학습 프레임

### 1.1 정의

*PAC (Probably Approximately Correct)* 학습: 학습기가 *충분히 많은 샘플* 을 받으면, *높은 확률* 로 *근사적으로 정확한* 가설을 출력합니다.

수학적으로:

$$P(\text{error}(h) > \epsilon) < \delta$$

- $\epsilon$: 정확도 (작을수록 좋음)
- $\delta$: 신뢰도 (작을수록 좋음)
- 필요 샘플 수: $N \geq f(\epsilon, \delta, \text{복잡도})$

### 1.2 의미

PAC 프레임은 *완벽한 가설* 을 보장하지 않습니다. *어느 만큼 정확하게·어느 만큼 자주* 의 두 차원으로 학습 성공을 정의합니다.

### 1.3 챕터 정리

PAC 학습은 *근사 정확성·확률적 보장* 의 두 차원으로 학습을 정의합니다.

---

## 2. VC 차원

### 2.1 정의

가설 클래스 $H$ 의 *VC 차원* 은 *H 가 임의로 라벨링할 수 있는 최대 점 수* 입니다.

예시:
- 1D 임계값 분류기: VC 차원 = 1
- 2D 선형 분류기: VC 차원 = 3
- d 차원 선형 분류기: VC 차원 = d + 1
- 신경망: 매우 높음 (파라미터 수에 비례)

### 2.2 일반화 한계

PAC 한계의 일반적 형태:

$$\text{generalization error} \leq O\left(\sqrt{\frac{\text{VC dim}}{N}}\right)$$

- VC 차원이 클수록 *일반화 오차 한계가 커짐*
- 샘플 수 $N$ 이 클수록 *일반화 오차 한계가 작아짐*

이 식은 *모델 복잡도와 데이터 양의 균형* 의 수학적 표현입니다.

### 2.3 챕터 정리

VC 차원은 *모델 클래스의 표현력* 을 측정하며, 일반화 오차 한계의 핵심 요소입니다.

---

## 3. 신경망의 미스터리

### 3.1 고전 이론의 한계

신경망은 VC 차원이 *데이터 샘플 수보다 훨씬 큰* 경우가 대부분입니다. 고전 이론에 따르면 *과적합으로 일반화 실패* 해야 합니다. 그러나 실제로는 *잘 일반화* 합니다.

### 3.2 현대 가설

- *암묵적 정칙화* — SGD 자체가 *간단한 가설* 로 편향
- *Lottery Ticket* — 큰 신경망 안에 *작은 효과적 부분망* 이 존재
- *Double Descent* — 모델 크기를 일정 수준 넘으면 *오차가 다시 감소*
- *Neural Tangent Kernel* — 무한 폭 한계의 분석

이 가설들은 *부분적 설명* 이며 *완전한 이론* 은 아직 없습니다.

### 3.3 챕터 정리

신경망 일반화는 *고전 이론으로 설명되지 않는 현상* 이며, 현재 연구 주제입니다.
