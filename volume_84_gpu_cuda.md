# Volume 93 — GPU 아키텍처와 CUDA

> 이 권이 끝나면 `nvidia-smi` 의 모든 숫자가 의미를 가지게 됩니다.

## 목적

GPU 는 단순히 *빠른 CPU* 가 아닙니다. 수만 개의 단순 코어를 동시에 돌리는 *대규모 병렬 처리기*이며, 이 본질을 이해하지 못하면 추론 최적화·분산 학습·메모리 최적화의 어떤 의사결정도 직관 없이 따라하게 됩니다. 이 권은 SM·워프·메모리 계층·Tensor Core·자료형까지를 *AI 엔지니어가 매일 마주치는 수준*으로 정리합니다.

## 학습 결과

이 권을 마치면 다음을 할 수 있게 됩니다.

1. CPU 와 GPU 의 설계 철학 차이를 한 문단으로 설명할 수 있습니다.
2. SM·워프·CUDA 코어·Tensor Core 를 구분할 수 있습니다.
3. HBM·L2·SMEM·레지스터의 대역폭·지연 차이를 알 수 있습니다.
4. FP32·FP16·BF16·FP8 의 표현 범위·정밀도·용도를 구분할 수 있습니다.
5. `nvidia-smi`·`nvtop`·`nsys` 의 각 화면이 무엇을 보여 주는지 압니다.
