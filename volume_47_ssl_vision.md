# Volume 47 — 자가지도 비전

> 이 권이 끝나면 *라벨이 없는 이미지로도 강력한 표현을 학습할 수 있다* 는 패러다임을 직관적으로 이해하게 됩니다.

## 목적

라벨링 비용이 비싼 이미지 데이터에서, *라벨 없이* 모델을 학습시키는 자가지도 학습 (SSL) 이 등장했습니다. SimCLR·MoCo·DINO·MAE 같은 알고리즘이 *ImageNet 1000 만 라벨 없이 사전학습* 후 다운스트림 작업에서 supervised 못지않은 성능을 보입니다.

## 학습 결과

1. 자가지도 학습의 *프리텍스트 작업 (pretext task)* 발상을 이해합니다.
2. SimCLR·MoCo 의 *대조 학습 (contrastive)* 메커니즘을 알 수 있습니다.
3. DINO·DINOv2 의 *Self-Distillation* 발상을 설명합니다.
4. MAE 의 *마스킹 + 복원* 패러다임을 이해합니다.

## 1. 프리텍스트 작업

라벨이 없으니 *데이터 자체에서 작업을 만듭니다*.

초기 사례:
- *이미지 회전 예측* — 0/90/180/270 도 중 어느 것?
- *조각 퍼즐 맞추기* — 9 조각의 위치 맞추기
- *컨텍스트 예측* — 주변 패치로 가려진 부분 예측

이 작업들이 *유용한 표현 학습* 을 유도합니다.

---

## 2. 대조 학습 (Contrastive Learning)

### 2.1 SimCLR (2020)

같은 이미지의 *두 다른 증강* 은 *비슷한 임베딩*, 다른 이미지는 *다른 임베딩* 이 되도록 학습.

```python
# 의사코드
for batch in loader:
    aug1 = strong_augment(batch)
    aug2 = strong_augment(batch)
    z1 = encoder(aug1)
    z2 = encoder(aug2)
    loss = contrastive_loss(z1, z2)   # InfoNCE
```

InfoNCE 손실 = *positive pair 는 가깝게, negative pair (다른 이미지) 는 멀게*.

### 2.2 MoCo (2019)

큰 negative pool 을 *큐로 유지* 해 SimCLR 의 큰 배치 요구를 완화.

---

## 3. Self-Distillation — DINO

DINO (2021) 와 DINOv2 (2023) 는 Meta 의 자가지도 학습 모델.

발상: *Teacher 와 Student 가 같은 이미지의 다른 뷰를 보고 같은 출력* 을 만들도록.

DINOv2 는 *14M 이미지 자가지도* 학습으로 supervised ImageNet 모델과 비슷한 성능 + *더 일반화*.

---

## 4. Masked Image Modeling — MAE

### 4.1 발상

이미지의 *75% 를 마스킹* 하고 *나머지 25% 로 마스킹된 부분 복원*. BERT 의 마스킹 LM 의 비전 버전.

```
원본 이미지 → 75% 패치 마스킹 → ViT 인코더 (보이는 25% 만) → 디코더 → 복원
```

장점: *큰 마스킹 비율* 덕분에 학습이 매우 효율적. 75% 가 마스킹되어도 75% 의 패치만 인코더에 들어가므로 *계산이 가볍다*.

### 4.2 결과

ImageNet 자가지도 사전학습 후 미세조정 시 *supervised 학습 능가*.
